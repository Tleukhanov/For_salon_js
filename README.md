# Салон красоты Бьюти — сайт записи

Веб-приложение для онлайн-записи в салон красоты. Клиенты могут выбрать мастера, услугу, дату и время. Администратор управляет мастерами, блокировками времени и просматривает записи.

## Стек

- Node.js
- Express
- SQLite (sql.js)
- vanilla HTML / CSS / JS

## Структура

```
forsalon/
├── server.js              # Точка входа
├── package.json
├── .env.example           # Шаблон переменных окружения
├── db/
│   ├── database.js        # Подключение к SQLite, создание таблиц
│   └── seed.js            # Наполнение тестовыми данными
├── middleware/
│   └── auth.js            # requireAdmin — проверка сессии админа
├── routes/
│   ├── public.js          # Публичное API (мастера, услуги, слоты, бронирование)
│   └── admin.js           # Admin API (логин, управление мастерами, блокировки)
└── public/
    ├── index.html          # Главная страница
    ├── booking.html        # Пошаговая запись
    ├── admin.html          # Панель администратора
    ├── css/style.css       # Стили
    └── js/
        ├── main.js
        ├── booking.js      # Логика бронирования
        └── admin.js        # Логика админки
```

## Запуск

```bash
# 1. Установить зависимости
npm install

# 2. Создать .env из образца
cp .env.example .env

# 3. Наполнить базу тестовыми данными
node db/seed.js

# 4. Запустить сервер
npm start
```

Сервер запустится на порту, указанном в `.env` (по умолчанию `3000`).

## Как поменять пароль администратора

Откройте `.env` и измените значение `ADMIN_PASSWORD`:

```
ADMIN_PASSWORD=новый_пароль
```

После изменения перезапустите сервер.

## Публичные страницы

- `http://localhost:3000/` — главная
- `http://localhost:3000/booking.html` — запись
- `http://localhost:3000/admin.html` — админка (пароль из `.env`)
